# 의사결정 로그 (Decision Log)

각 분기에서 **권장안으로 자동 진행**한 결정을 기록합니다. 나중에 한 번에 검토/재결정하세요.
형식: `결정 / 선택한 권장안 / (대안) / 이유`

| # | 결정 | 확정값 |
|---|---|---|
| P1 | 테넌시 | 공유 멀티테넌시(테넌트 격리·추출 가능), 신원도 테넌트별 |

| P4 | 자연어→템플릿 | Claude로 자연어 매뉴얼→그래프 변환 |
| P5 | 그래프 편집 | **그래프가 원본**, 사용자 수정본이 최종. 자연어는 생성 도구 |
| P6 | 조직도 | 수동 입력(부서/직급/보고선/전결규정) |
| P7 | 결재자 지정 | 결재=조직도 상대 지정 기본(+고정 옵션) / 작업=템플릿 구동 시 일괄 지정 |

| P10 | 목표 연결 | OKR/KPI 연결은 선택, 독립 프로세스 허용 |
| P11 | 업무 지시 | 누구나 가능(조직도 제한 옵션) → 재작업(rework) 과제 생성 |
| P12 | rework | TASK 반복 수행 허용 |
| P13 | OKR/KPI | 정성=OKR, 정량=KPI 둘 다 / 정식 케스케이딩(회사→부서→개인) |

| P15 | 사후 프로세스 수정 | 협조업무 실행 중 추가 → 프로세스 승인체계로 승인 → 병렬/인라인 선택 |
| P16 | 분석 | 업무 흐름 병목·사이클타임·처리량·재작업률 가시화 |
